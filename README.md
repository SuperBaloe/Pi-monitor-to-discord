# Pi Monitor to Discord

## Libraries

### Built-in
- `os`
- `time`
- `json`
- `datetime`
- `socket`

### Installed
- `requests`
- `psutil`
- `questionary`

## Features

Posts your machine CPU and RAM usages and uptime to Discord chat.

The whole program can be used in the terminal and automatically creates its own config and message ID file.
